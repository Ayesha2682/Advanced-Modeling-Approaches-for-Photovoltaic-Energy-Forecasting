# Photovoltaic Power Forecasting using Graph Learning, Multi-Task Modeling, and Spiking Transformers

## 📌 Overview

This repository contains the implementation of the models proposed in my MSc thesis:

**"Advanced Deep Learning Approaches for Photovoltaic Power Forecasting"**

The goal of this work is to improve photovoltaic (PV) power forecasting by addressing three key challenges:
- Modeling **spatial dependencies**
- Capturing **system-level interactions (generation & consumption)**
- Improving **temporal modeling efficiency**

To achieve this, three complementary frameworks are developed:
1. **Spatiotemporal Graph Learning Framework**
2. **Multi-Task Generation and Consumption Forecasting Framework**
3. **Event-Driven Spiking Transformer Framework**
